# iii OpenGL Visualizer PoC

This is a Proof-of-Concept (PoC) OpenGL visualizer for the `iii` (Implementation is Illustration) library. It demonstrates how to decouple simulation code from visualization by recording events and playing them back in a separate interactive environment.

## Features

-   **Event-Based Playback**: Visualizes traces generated by `iii` code (`Vector3`, `Matrix4`).
-   **Interactive Timeline**: Play, pause, scrub through time steps.
-   **Camera Control**:
    -   **Automatic**: Follows the camera movements scripted in the trace.
    -   **Manual**: Orbit (Right Mouse) and Zoom (Scroll) to inspect the scene.
-   **Live Parameters**: Tune simulation variables (like `dt`) in real-time using the **Parameters** window.
-   **Legend**: View object labels and colors.
-   **Demos**: Includes "Matrix Demo" (coordinate frames) and "Solar System" (orbits).

## Building

This project uses CMake. Ensure you have CMake and a C++20 compiler installed.

```bash
mkdir build
cd build
cmake ..
cmake --build . --config Release --target iii_visualizer
```

## Running

Run the executable from the build directory:

```bash
./Release/iii_visualizer.exe
```

## Usage

1.  **Demos**: Use the `Demos` menu to switch between `Matrix Demo` and `Solar System`.
2.  **Playback**: Use the timeline controls at the bottom left.
3.  **Camera**: Right-click and drag to orbit. Scroll to zoom. The camera gently transitions back to the script when you release control.
4.  **Parameters**: Adjust the `dt` slider in the Parameters window to change the speed of the Solar System simulation interacting with the playback.

## Architecture

-   `VisualizerApp`: Manages UI, Input, Camera state, and trace playback.
-   `GLRenderer`: Handles raw OpenGL rendering of `iii::Event` primitives.
-   `TestCases.hpp`: Contains the demo generation logic (simulation).
